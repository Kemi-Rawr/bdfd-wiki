# $botNode
Retorna o ID da node do bot.

## Sintaxe
```
$botNode
```
> Ver lista de Nodes e Status, [Clique aqui](https://botdesignerdiscord.com/status).

## Exemplo
```
$nomention
O ID da node do bot: $botNode
```
