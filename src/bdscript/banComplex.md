# $ban[]
Bane o usuário mencionado com um motivo.

## Syntax
```
$ban[Motivo]
```

### Parâmetros 
- `Motivo` `(Tipo: String || Indicação: Esvaziável)`: O motivo para o banimento, que vai ser salvo no registro de auditoria.

## Exemplo
```
$nomention
$ban[$noMentionMessage]
<@$mentioned[1]> foi banido!
```
