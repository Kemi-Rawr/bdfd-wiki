# $botCommands
Retorna uma lista dos comandos do bot.

## Sintaxe
```
$botCommands[Separador]
```

### Parâmetros
- `Separador` `(Tipo: String || Indicação: Obrigatório)`: Vai ser usado para separar cada comando.

## Exemplo
```
$nomention
$botCommands[, ]
```
